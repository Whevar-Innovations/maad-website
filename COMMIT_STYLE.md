# Commit Convention

All commits to this project must follow the convention below:

`[:emoji]: [status]: [Message]`

Then a blank line, followed by flat bullet points describing what changed.

### Status Types
- `:sparkles: Feat:`: For new features or enhancements.
- `:bug: Bug:`: For bug fixes.

### Rules
- Keep the summary imperative and concise.
- Capitalize the first word of the summary.
- Do not add AI attribution or co-author lines.

### Example
```text
:sparkles: Feat: Add smooth scrolling with Lenis

- Install lenis dependency.
- Create useSmoothScroll hook.
- Initialize Lenis in the Layout component.
```
